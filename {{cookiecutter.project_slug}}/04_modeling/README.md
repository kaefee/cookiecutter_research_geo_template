# Fase: Modeling

Esta sección contiene todos los componentes relacionados con el desarrollo y ajuste de modelos predictivos.

- `models/`: Implementaciones de arquitecturas y componentes principales.
- `training/`: Scripts y rutinas para entrenamiento, evaluación y ajuste de modelos.
- `experiments/`: Flujos experimentales, comparación de variantes y registración sistemática de resultados.
- `configs/`: Archivos de configuración reproducibles para la gestión de experimentos.
